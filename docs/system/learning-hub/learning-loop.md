# Learning Loop

Summary: The learning loop captures evidence, computes reusable features, and feeds quality intelligence into control-plane and autonomous prioritization.

## Storage
- `learning_evidence_events`
- `learning_features`
- `learning_metric_snapshots`
- `learning_drift_events`
- `learning_eval_labels`

## Feature examples
Supplier/shipping/stock reliability, parser yield trends, failure signatures, and quality regressions.

## Feedback reuse
- Control-plane scorecards
- Supplier prioritization and manual-review focus
- Drift-informed pause recommendations
