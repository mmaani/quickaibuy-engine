# Recovery Playbooks

Summary: Recovery emphasizes auto-heal for known integrity issues and explicit human actions for unresolved uncertainty.

## Auto-heal
- orphan/detached listing cleanup
- stale in-progress publish containment
- integrity reconciliation

## Operator actions
- refresh weak evidence
- investigate critical drift signatures
- resolve manual purchase/tracking exceptions

## Never override
- fail-closed shipping/stock/profit safety blocks
- guarded publish requirements
