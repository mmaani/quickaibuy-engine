# Attribute Intelligence

Summary: Attribute intelligence learns which structured specifics matter most by category and profile and exposes required, high-value, optional, and noise attributes.

## Purpose
- Improve publishability by category-aware specifics coverage.
- Show which missing attributes correlate with failure.
- Reduce low-value attribute noise.

## Inputs
- Listing payload item specifics
- Verified AI pack specifics
- Marketplace aspects
- Listing outcomes
- Product-market taxonomy

## Outputs
- Required attributes
- High-value attributes
- Optional attributes
- Noise attributes
- Coverage ratios
- Publish-success deltas when present vs missing

## Automation
- Listing generation guidance
- Review prioritization
- Candidate filtering and opportunity explanation

## AI Boundaries
- AI can normalize and rank attributes.
- Attribute truth still has to come from supplier or marketplace evidence.
